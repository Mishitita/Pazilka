¿Qué es el Diseño y Desarrollo Web?


No se confunda el Diseño Web con el Desarrollo Web, ya que son dos cosas diferentes. Aquí haremos un resumen de los dos a continuación.

Diseño Web

 El Diseño web es el encargado de dar un aspecto a una página web, ya sea de forma visual o con las interacciones que hace el visitante al entrar a la página. También puede ser nombrado como Interfaz de Usuario o "front end".
