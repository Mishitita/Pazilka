¿Qué Es Pazilka?


Pazilka o "P-CHAN", es un proyecto de un alumno de una institucion educativa de nivel media-superior. El trabajo inició como un proyecto que llevaría aproximadamente un semestre, con clases de HTML, aunque en este trabajo se agregaron otros lenguajes, como CSS y un poco de JavaScript.
Aún trato de agregar PHP a la página principal, pero no sé absolutamente nada de este lenguaje.
